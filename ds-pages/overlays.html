Usage Guidelines — Do's & Don'ts
Do — Modal
Use para ações críticas (excluir, confirmar, decisões irreversíveis). Mantenha o conteúdo conciso e forneça botões claros de cancelar/confirmar.
Don't — Modal
Evite modais para conteúdo informativo simples. Não empilhe modais. Use overlay escuro (rgba) para destacar o contexto.
Do — Tooltip
Use para dicas curtas (1–2 linhas). Garanta acessibilidade (hover + focus). Posicione de forma que não cubra o elemento alvo.
Don't — Tooltip
Não use para conteúdo essencial. Evite tooltips em elementos móveis. Não coloque informações críticas apenas em tooltips.
Do — Toast
Use para feedback de ações (salvar, enviar, erro). Auto-dismiss após 3–5s. Empilhe verticalmente quando houver múltiplos.
Don't — Toast
Não use para erros críticos que exigem ação. Evite toasts excessivos. Não coloque mais de 3 toasts visíveis ao mesmo tempo.
